## ⚙️ Instalação

```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/Hugovarellaa/dtmoney-2022
```

```
# Acesse a pasta da aplicação
cd dtmoney-2022

# Instale as dependências
$ yarn

# Inicie a aplicação back-end
$ yarn dev
```
